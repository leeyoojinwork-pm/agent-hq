# ROADMAP — Agent HQ

PM: **PM Fable** (5F Control). 이 문서가 프로젝트의 단일 진실 소스(SSOT)다.
누가 이어받든(Fable → Opus → 누구든) 여기의 체크박스 상태부터 읽고 시작한다.

## 프로덕트 비전

"내 에이전트들이 한 공간에서 일하고 있다"는 감각을 주는 개인 관제실.
대시보드가 아니라 **공간**이다. 숫자 나열이 아니라 야근 중인 빌딩을 보는 재미가 핵심 가치.

## 운영 규칙 (PM 노트)

1. **마일스톤 순서 고정.** M1 끝나기 전에 M2 코드 금지. 반쯤 붙은 실데이터가 최악.
2. 커밋 단위 = 체크박스 단위. 커밋 메시지에 `M1:` 프리픽스.
3. 디자인 회귀 금지 — 스크린샷 비교로 확인 후 머지.
4. 스코프 추가 요청은 여기 Backlog에 먼저 적고, 현재 마일스톤에 끼워넣지 않는다.

## 핸드오프 프로토콜 (Fable → Opus)

토큰/컨텍스트 소진으로 교대할 때:

1. Opus는 `docs/OPUS-BRIEF.md`의 복붙용 프롬프트로 시작한다
2. 이 파일의 체크박스에서 **첫 번째 미완료 항목**이 현재 작업이다
3. 진행 중이던 작업은 커밋되어 있거나, 커밋 불가면 이 파일 하단 "인계 메모"에 상태를 남긴다
4. 교대 후 첫 행동: 체크박스 상태와 리포 실제 상태가 일치하는지 검증
